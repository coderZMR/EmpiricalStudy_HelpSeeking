# Empirical-Study
程序员在什么情况下需要什么样的帮助
## RQ 1.程序员什么时候不会
## RQ 2.程序员为什么不会
## RQ 3.程序员在不会的时候需要什么样的帮助
### 例如：给程序员推荐API，给程序员API的使用文档，给程序员推荐代码样例
## RQ 4.通过何种交互方式给程序员提供帮助
### 例如：编程助手根据程序员提出的问题进行一次对话或多次对话以推荐出目标结果，编程助手根据程序员当前所要实现的需求或代码上下文环境进行预测


## 实验思路
先做实验室的学生结对编程的经验研究实验，然后对该实验结果进行分析，对以上四个RQ的答案进行分析得出初步结论，之后由该结论形成调查问卷，通过调查问卷对更大范围的程序员进行调研。
先做实验室的学生结对编程实验的目的是对后面设计调查问卷提供依据（问卷中题目答案的选项设计）。
